GitHub+Hexo搭建个人博客辛酸史--基础搭建(壹)

发表于2016-06-06|个人博客辛酸系列
|阅读量:

概述

hexo是一款基于Node.js的静态博客框架, Hexo github 链接 链接 ,这篇教程是针对与Mac的，参考链接，由于原文讲到的hexo是以前的老版本，所以现在的版本配置的时候会有些改动。

PS : 以下教程是针对hexo的版本为3.x

准备工作

  • 安装node

    到 Node.js 官网下载相应平台的最新版本，一路安装即可。

  • 安装Git

    下载 msysgit 并执行即可完成安装。

  • 安装hexo

    使用以下命令安装hexo到全局（在任意位置点击鼠标右键，选择Git bash） npm config set registry http://registry.cnpmjs.org```

  • 创建hexo文件夹

    安装完成后，在你喜爱的文件夹下（如H:\hexo），执行以下指令(在H:\hexo内点击鼠标右键，选择Git bash)，Hexo 即会自动在目标文件夹建立网站所需要的所有文件。 ssh-keygen -t rsa -C “你的邮箱地址”```
    按3个回车，密码为空。
    在C:\Users\Administrator.ssh下，得到两个文件id_rsa和id_rsa.pub。

  • 在GitHub上添加SSH密钥

    打开id_rsa.pub，复制全文。https://github.com/settings/ssh ，Add SSH key，粘贴进去。

Hexo简单介绍

  • 目录结构

    hexo g -d```

    hexo deploy问题：Deployer not found: git

    title: #文章的标题
    date: 2015-07-30 07:56:29 #发表日期，一般不改动（可以改动）
    categories: categories1 #文章文类

    tags: [tags1,tags2] #文章标签，多于一项时用这种格式

    正文，使用Markdown语法书写```

    域名的绑定

  • 创建CNAME文件

    在站点source目录或者在Repository的根目录下面，新建一个名为CNAME的文本文件，里面写入你要绑定的域名，比如dinson.win

  • DNS域名指向

    在你购买域名的网站的控制台修改，例如我在阿里云买的，则去阿里云登录，再把你的域名指向你项目的IP地址，如何知道你项目的ip地址呢，在命令提示符中输入下面代码
      ping "你的用户名".github.io
